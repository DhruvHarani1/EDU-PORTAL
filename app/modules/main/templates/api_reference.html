API v1

Authentication

EduPortal uses API keys to authenticate requests. You can view and manage your API keys in the developer dashboard.

POST /auth/login

Authenticate a user and retrieve an access token.

curl -X POST https://api.eduportal.com/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@example.com",
    "password": "yourpassword"
  }'
GET /students/{id}

Retrieve details of a specific student.

{
  "id": "STU-2024-001",
  "name": "Alex Johnson",
  "email": "alex@university.edu",
  "enrollment_date": "2024-09-01",
  "status": "active"
}
